# 玉米期货量化决策系统

基于 DCE 玉米期货 21 年历史数据（2004-2026），集成 LightGBM、LSTM、Prophet 三类机器学习模型，提供多周期（1日/5日/20日）价格预测与方向信号。

## 数据来源

| 文件 | 说明 |
|---|---|
| `1-量价/corn_en_no_oi.txt` | DCE 玉米期货日线数据（2004-2026，5246行） |
| `2-基本面/国内玉米基本面日度数据.xlsx` | 基本面数据（2014-2026，2999行，有效区间 2020+） |

## 系统架构

```
量价特征工程 ──→ LightGBM ──┐
LSTM 序列 ──────→ LSTM ────┤──→ 模型集成 ──→ 综合预测信号
基本面特征 ───────────────┘
(Regime: 2020+ 全量 / 2014-2020 部分 / 2004-2013 仅量价)
```

## 模型说明

| 模型 | 数据范围 | 擅长 |
|---|---|---|
| **LightGBM** | 2004-2026 全量 | Tabular 特征融合，量价+基本面特征 |
| **LSTM** | 2004-2026 全量 | 序列趋势捕捉 |
| **Prophet** | 全量+基本面 | 时序分解，周期发现 |

## 启动方式

```bash
cd 玉米决策系统
pip install -r requirements.txt
python app.py
```

打开浏览器访问 http://localhost:8000

## API 接口

| 端点 | 说明 |
|---|---|
| `GET /api/health` | 系统健康状态 |
| `GET /api/prediction` | 最新多模型预测 |
| `GET /api/price` | 量价数据 + 技术指标 |
| `GET /api/fundamental` | 基本面数据 |
| `GET /api/summary` | 数据质量报告 |
| `GET /api/features` | 特征列表 |

## 自定义金融逻辑规则

编辑 `signals.py`，在 `evaluate_all_rules()` 函数中添加你自己的交易规则。系统会自动检测模型预测与人工规则的方向背离并预警。
